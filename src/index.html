What I’m all about

I enjoy doing full-stack web development. A non-exhaustive list of examples:

  • Configuring NGINX to serve static files, reverse-proxy apps, and rate-limit
  • Creating RESTful JSON APIs to be consumed by front-end frameworks like Vue.js
  • Using the latest ES6+ JavaScript features in Node.js or (transpiled) in a browser
  • Use automation tools like Gulp to bundle code & assets in a production-ready pipeline
  • Writing SCSS using features like Flexbox & CSS Grid all piped through Autoprefixer

My interests aren’t limited to the aforementioned, though. I’m always interested in how I can automate things—especially boring, repetitive things. Testing the things I architect and write proves a vital yet challenging and sometimes mundane task. Recently, “snapshot testing” has piqued my interest.
